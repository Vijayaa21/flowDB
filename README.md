FlowDB is an open-source, git-native database branching tool designed for teams building on Vercel.
This monorepo contains the core fork engine, CLI, reconciler, SDK, orchestration API, dashboard, and docs surfaces.
The current scaffold focuses on workspace structure and shared developer tooling, with feature code intentionally omitted.
